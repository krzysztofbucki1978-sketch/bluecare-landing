⚠️ TOP 5 BŁĘDÓW & ROZWIĄZANIA
══════════════════════════════

1. "npm install" error
   → Pobierz Node.js z https://nodejs.org/
   
2. "Build failed" na Vercel
   → Sprawdź Vercel Logs → napraw → git push
   
3. Formularz nie wysyła
   → Sprawdź Formspree ID w ContactForm.jsx
   
4. Meta-tagi nie wyświetlają się
   → Czekaj 1-2 min (cache) lub private/incognito mode
   
5. Port 3000 już zajęty
   → npm run dev -- -p 3001
