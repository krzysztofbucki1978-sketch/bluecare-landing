1. **VSCode Extensions**
   - Tailwind CSS IntelliSense
   - ES7+ React/Redux/React-Native
   - Prettier Code Formatter

2. **Terminal shortcuts**
   - `npm run dev` - start dev server
   - `npm run build` - test production build
   - `git status` - sprawdź changed files

3. **Deploy process**
   - Merge na `main` branch = auto-deploy na Vercel
   - Nie czekaj deployment statusu - może trwać 2-3 minuty
